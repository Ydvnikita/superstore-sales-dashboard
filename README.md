superstore-sales-dashboard


Project Overview

This project focuses on Data Visualization and Storytelling using the Sample Superstore dataset. The dashboard was created to analyze sales performance, profitability, customer segments, and regional trends.

Objective

To transform raw sales data into meaningful business insights through interactive visualizations and storytelling techniques.

Dataset

- Sample Superstore Dataset
- Format: CSV
- Records: Sales transactions across multiple categories, regions, and customer segments

Tools Used

- Power BI
- Microsoft Excel (Data Review)
- GitHub

Dashboard Components

1. Sales Trend Analysis

- Monthly sales performance
- Identification of peak sales periods

2. Category Analysis

- Sales comparison across categories
- Identification of top-performing categories

3. Regional Analysis

- Revenue contribution by region
- Regional performance comparison

4. Customer Segment Analysis

- Consumer
- Corporate
- Home Office

5. Profitability Analysis

- Profit by category and sub-category
- High-profit and low-profit products

Key Insights

- Total Sales: $2.29 Million
- Total Profit: $286K
- Technology category generated the highest sales.
- West region contributed the highest revenue.
- Consumer segment was the largest contributor to sales.
- November recorded the highest monthly sales.
- Copiers and Phones were among the most profitable sub-categories.

Recommendations

- Increase investment in Technology products.
- Strengthen marketing efforts in high-performing regions.
- Improve profitability of low-margin products.
- Enhance customer retention strategies.
